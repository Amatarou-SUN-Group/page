Console Escape Sequence Functions Pack - C++


About


A header file that makes it easy to qualify the console with escape sequences.
It currently supports normal colors (8 colors), extended colors (256 colors), bold, italic, and underline.

List of features


[Variables and constant values]
CON_RED : Console Color (8 color) - Red
CON_YELLOW : Console Color (8 color) - Yellow
CON_GREEN : Console Color (8 color) - Green
CON_CYAN : Console Color (8 color) - Cyan
CON_BLUE : Console Color (8 color) - Blue
CON_MAGENTA : Console Color (8 color) - Magenta
CON_WHITE : Console Color (8 color) - White
CON_BLACK : Console Color (8 color) - Black
CON_DEFAULT : Console Color (8 color) - Default color

nowfore : (int) Now foreground color
nowback : (int) Now background color

extended_color_b : (bool) Is Background Color Extended
extended_color_f : (bool) Is Foreground Color Extended

bold : (bool) Is Bold enabled (get, set)
italic : (bool) Is Italic enabled (get, set)
underline : (bool) Is Underline enabled (get, set)
st : (bool) Is StrikeThrough enabled (get, set)

[Functions]
applystyle() : Apply Styles and Colors
Set_Forecolor(color) : Set Foreground Color + Apply
Set_Backcolor(color) : Set Background Color + Apply
Set_ForecolorEx(color) : Set Foreground Color (Extended Color 256 colors) + Apply
Set_BackcolorEx(color) : Set Background Color (Extended Color 256 colors) + Apply
Set_Bold(bool) : Enable/disable Bold + Apply
Set_Italic(bool) : Enable/disable Italic + Apply
Set_Underline(bool) : Enable/disable Underline + Apply
Set_St(bool) : Enable/disable StrikeThrough + Apply
Set_Styles(bold, italic, underline, st) : Style it all at once + Apply
Reset_Forecolor() : Reset Foreground Color + Apply
Reset_Backcolor() : Reset Foreground Color + Apply
Reset_Colors() : Reset Foreground and Background Color + Apply
Reset_Styles() : Reset Styles + Apply
Reset_All() : Reset All (Styles and Colors) + Apply
Set_CursorPos(x, y) : Set Console Cursor Position by absolute coordinate
Clear_Console() : Clear the Console
putstatus_to_console() : Output Color and Style Status to the Console
