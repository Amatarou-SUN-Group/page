How to run Forge installer on ubuntu


writer : Amatarou-SUN

Everyone wants to install mods on ubuntu and play minecraft, right?
But I think that there is a point where it gets caught a little, so I would like to write a little about how to deal with it.
Note: I'm beginner to Linux. Please self-responsibility.

Up to the point of downloading Forge's jar, there is no problem normally.
But I get this error when I try to run the downloaded file.

This error can be resolved by marking the forge jar as executable.
However, it is not possible in such a GUI environment, so we will use the CUI environment.

Use the chmod command to change permissions and marks.
By the way, the mark before mark operation is like this.
-rw-rw-r--

By doing this, you can run the Forge installer normally
