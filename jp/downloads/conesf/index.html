エスケープシーケンス関数パック - C++


概要


簡単にコンソールをエスケープシーケンスで修飾できるようにするヘッダファイルです。
現在 標準色 (8色), 拡張色 (256色),太字, 斜字,下線に対応しています。

[注意]
まだ完成していません。
不完全です。

機能一覧


[変数, 定数]
CON_RED : コンソールの色 (8色) - 赤
CON_YELLOW : コンソールの色 (8色) - 黄色
CON_GREEN : コンソールの色 (8色) - 緑色
CON_CYAN : コンソールの色 (8色) - 水色
CON_BLUE : コンソールの色 (8色) - 青色
CON_MAGENTA : コンソールの色 (8色) - 紫色
CON_WHITE : コンソールの色 (8色) - 白色
CON_BLACK : コンソールの色 (8色) - 黒色
CON_DEFAULT : コンソールの色 (8色) - 既定色

nowfore : (int) 今の文字色
nowback : (int) 今の文字背景色

extended_color_b : (bool) 背景色の色は拡張されてるか?
extended_color_f : (bool) 文字色の色は拡張されてるか?

bold : (bool) 太字は有効か? (取得, 設定)
italic : (bool) 斜字は有効か? (取得, 設定)
underline : (bool) 下線は有効か? (取得, 設定)
st : (bool) 取消線は有効か? (取得, 設定)

[関数]
applystyle() : スタイル・色を適用
Set_Forecolor(color) : 文字色を設定 + 適用
Set_Backcolor(color) : 文字背景色を設定 + 適用
Set_ForecolorEx(color) : 文字色を設定 - 色拡張 (256色) + 適用
Set_BackcolorEx(color) : 文字背景色を設定 - 色拡張 (256色) + 適用
Set_Bold(bool) : 太字の有効・無効を設定 + 適用
Set_Italic(bool) : 斜字の有効・無効を設定 + 適用
Set_Underline(bool) : 下線の有効・無効を設定 + 適用
Set_St(bool) : 取消線の有効・無効を設定 + 適用
Set_Styles(bold, italic, underline, st) : スタイルの一気設定 + 適用
Reset_Forecolor() : 文字色をリセット + 適用
Reset_Backcolor() : 文字背景色をリセット + 適用
Reset_Colors() : 文字色, 文字背景色をリセット + 適用
Reset_Styles() : スタイルのリセット
Reset_All() : 色, スタイルをすべてリセット
Set_CursorPos(x, y) : カーソルの位置を絶対座標で指定 (横 : x, 縦 : y)
Clear_Console() : コンソールをクリア
putstatus_to_console() : 現在の情報をコンソールに出力
