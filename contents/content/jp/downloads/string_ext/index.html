String ExTプロジェクト

C#のStringの機能を拡張に追加します。
メインのクラスはStringExTです。

Replace_Class

----string 置き換え前 ----string 置き換え後

String ExT

----[戻り型 : string] Replace () : 置き換え前の文字列を置き換え後の文字列に置き換え
----[戻り型 : string] iReplace : 置き換え後の文字列を置き換え前の文字列に置き換え
----[戻り型 : string] Replace_List : 配列で置き換え
----[戻り型 : string] iReplace_List : 配列で逆置き換え
